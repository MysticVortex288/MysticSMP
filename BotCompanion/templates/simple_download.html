Installations-Anleitung

  1. Entpacke die ZIP-Datei in einen neuen Ordner
  2. Erstelle eine .env Datei mit folgenden Informationen:
    3. DISCORD_TOKEN=dein_discord_bot_token DISCORD_CLIENT_ID=dein_discord_client_id DISCORD_CLIENT_SECRET=dein_discord_client_secret
  3. Installiere die benötigten Pakete:
    4. pip install -r packages_list.txt
  4. Starte den Bot:
    5. python main.py

Wichtige Dateien

main.py - Hauptdatei des Bots
bot.py - Bot-Konfiguration und Initialisierung
app.py - Web-Dashboard
cogs/ - Ordner mit allen Funktionsmodulen
templates/ - HTML-Vorlagen für das Dashboard
static/ - Statische Dateien (CSS, JS, Bilder)
utils/ - Hilfsfunktionen
*.json - Konfigurationsdateien
